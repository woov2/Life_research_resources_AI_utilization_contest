README.MD (additional_data)

- additional_data folder
1.  https://portal.gdc.cancer.gov/analysis_page?app= 사이트 접속
2. 사이트 접속후 Search 창에 TCGA-암종류 검색후 SAVE NEW COHORT 눌러서 COHORT 에 저장
ex) TCGA-BLCA 검색후 'TCGA-BLCA' 이름으로 SAVE NEW COHORT 눌러서 COHORT 저장
3. https://portal.gdc.cancer.gov/analysis_page?app= 사이트의 추가된 COHORT 를 선택함
4. 선택된 COHORT 의 Cohort Level MAF 클릭후 스크롤바 를 끝까지 내려서 우측 하단에 download 클릭
5. 이 과정을 additional_data 에 있는 암종에 일일이 처리해줌.

- additional_data2 folder
1. https://www.cbioportal.org/datasets 사이트 접속
2. 각 암에 맞는 데이터 파일 매칭 후 데이터 다운로드 후 폴더에 개별 저장.
ex) ucec -> Uterine Corpus Endometrial Carcinoma (TCGA, Firehose Legacy)
            -> Uterine Corpus Endometrial Carcinoma (TCGA, Nature 2013)
            -> Uterine Corpus Endometrial Carcinoma (TCGA, PanCancer Atlas)
데이터 다운로드해서 UCEC 폴더에 넣어줌.
